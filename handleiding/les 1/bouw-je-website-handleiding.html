Bouw je eigen Website! - Les 1

Wat je nodig hebt

  • Een laptop
  • Een tekstverwerker waarin je de code kan schrijven. Wij raden Notepad++ aan.
  • Een internetbrowser waarin je je website kan bekijken.

De voorbereiding

Laten we de webpagina openen in je internetbrowser om te zien hoe het er nu uitziet.

  • Ga naar de mijn-eerste-website map
  • Open het mijn-website.html bestand door er met je linkermuisknop op te dubbelklikken.
De webpagina ziet er nu zo uit: Nog niet zo mooi, hè? Maar dat gaan we veranderen!

Om de website te kunnen aanpassen moeten we de webpagina in een tekstverwerker openen.

  • Dit doe je door met je rechtermuisknop op mijn-website.html te drukken...
  • ...dan op "Openen met" te gaan staan met je muis...
  • ...en dan op "Notepad++" te drukken.

1. Je eigen titel

Je eigen website heeft natuurlijk een eigen titel nodig! Laten wij er een toevoegen.

Als het goed is zie je in Notepad++ nu deze tekst voor je: 

<!DOCTYPE html>
<html>
  <head>
    <title> </title>
    <link rel="stylesheet" type="text/css" media="screen" href="stijl.css" />
  </head>
  <body>
    <div id="pagina">
      <header>
        <h1> Mijn website </h1>
      </header>
      <main>
        
      </main>
      <footer>

      </footer>
    </div>
  </body>
</html>
Verwijder nu Mijn website en zet er de naam van jouw nieuwe website neer.
Mijn website gaat over robots, dus ik noem hem Robobots
Als je nu weer naar je internetbrowser gaat en op "refresh" drukt (Ctr R, F5 of op de ronde-pijl-knop) kun je de nieuwe naam van je website zien!

1.1. Wat is er gebeurd?

De .html in de bestandsnaam zegt tegen de computer dat het een website is.
De internetbrowser leest vervolgens de tekst en vertaalt het naar "elementen" die het aan jou laat zien.

Een element bestaat uit drie dingen:

  • De "opening tag", zoals <h1>
  • De inhoud, zoals Mijn website
  • De "closing tag", zoals </h1>

De "opening tag" zegt tegen de browser welk element hij moet laten zien. In dit geval een grote vette kop ("h1").
De "closing tag" zegt tegen de browser dat dit het einde van het element is.
Deze tag is bijna hetzelfde als de opening tag, maar dan met een / (na de < en voor de h1).
Alles wat er tussen staat is de inhoud. Dit is nu "Mijn website".
Nu weet de browser dat hij "Mijn website" groot en vet moet laten zien!

2. Iets om te lezen

Een grote kop is mooi, zo weten de bezoekers de naam van jouw website.
Maar, we willen de bezoeker ook iets laten lezen!
Daar gaan wij nu voor zorgen.

Voeg, tussen de <main> en </main> tags, deze code toe: 

<p> Deze website gaat over robots. Robots zijn echt supercool. </p>
Kies een inhoud die je leuk lijkt en bij jouw website past.
Ik heb gekozen voor een tekst over robots, omdat mijn website daarover gaat.

2.1. Wat is er gebeurd?

Precies wat je voor de vette kop hebt gedaan, maar dan iets anders.
Door <p> en </p> te gebruiken heb je tegen de browser gezegd dat hij een paragraaf (een stukje tekst) moet laten zien.
Deze laat de browser op een andere manier zien dan de kop: kleiner en minder vet.

3. Je website mooi maken!

Nu heb je wat tekst op je webpagina, maar echt mooi ziet het er nog niet uit...
Laten wij de website een crazy coole stijl geven!

Open stijl.css in Notepad++, deze staat in dezelfde map als mijn-website.html

  • Tip: gebruik dezelfde stappen als in het openen van mijn-website.html!
Dit bestand is nu leeg. Kopieer en plak deze code in het lege bestand: 
body { 
  background: gold;
}
 Als je nu je het CSS bestand opslaat (Ctr S) en de webpagina in je browser refreshed zie je dat de achtergrond in goud is veranderd!
Je kan ook een andere kleur kiezen. Verander dan gold in bijvoorbeeld:
firebrick deeppink dodgerblue darkviolet green orange lightpink
Op deze website staan nog veel meer kleuren waar je uit kan kiezen!

Laten we de tekst ook wat leuker maken.
Voeg deze regel code toe na background: gold;: 

font-family: 'Segoe UI', Tahoma, Geneva, Verdana, sans-serif;
Nu zou je code in stijl.css er zo uit moeten zien: 
body { 
  background: gold;
  font-family: 'Segoe UI', Tahoma, Geneva, Verdana, sans-serif;
}
 Als je nu naar je website gaat en refresht zou je tekst er anders uit moeten zijn.

3.1 Wat is er gebeurd?

De code hier boven ziet er anders uit dan je gewend bent, toch?
Dat is omdat dit niet in HTML is geschreven, maar in een taal die CSS heet.
Deze taal wordt gebruikt om websites mooi te maken.

Ergens boven in mijn-website.html zie je deze regel code staan: 

<link rel="stylesheet" type="text/css" media="screen" href="stijl.css" />
Deze regel zegt de internetbrowser dat hij de code in stijl.css moet gebruiken om de website mooi te maken.

3.1.1. CSS

Een CSS-bestand bestaat uit blokken van CSS-regels.

Elk blok begint met een "selector".
Deze zegt de browser op welke html-elementen hij de volgende regels toe moet passen.
In ons geval zijn dit alle body elementen. Dit is er maar één.

De code tussen de haakjes { } zijn de CSS-regels.
De eerste regel zegt dat de achtergrond ("background" in Engels) van de body goud moet zijn.
De tweede regel zegt dat de tekst een bepaalde vorm (een lettertype of font) moet hebben.

Elke CSS-regel bestaat uit:

  • Een eigenschap die veranderd moet worden (zoals background)
  • Een dubbele punt :
  • Een waarde: wat de eigenschap moet zijn, bijvoorbeeld gold
  • Een punt-komma ;

Er zijn heel veel eigenschappen die je met CSS aan kan passen!
We gaan er in de volgende handleiding nog een paar zien.
