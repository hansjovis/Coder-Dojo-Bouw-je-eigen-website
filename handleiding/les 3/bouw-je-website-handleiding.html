Let op: je website ziet er nu anders uit!

1. Een link toevoegen.

Voeg deze code toe aan je webpagina: 

<a href="https://nl.wikipedia.org/wiki/Robot"> robots </a>
Deze link zal er zo uit zien:
robots

2. Wat is er gebeurt?

Een link komt meestal in de vorm van een a-element.
Deze heeft minstens twee onderdelen:

  • Een href-attribuut. (Geel)
    Hierin moet je het adres van de webpagina invullen waar je naar wil verwijzen.
    Deze kan je vinden in de adresbalk van de browser (de brede witte balk aan de bovenkant).
    In dit geval verwijst de link naar https://nl.wikipedia.org/wiki/Robot
    Maar je mag natuurlijk een andere webpagina kiezen ðŸ˜ƒ !
    Tip: Vergeet niet " om het adres te zetten!
  • Een link-tekst. (Blauw)
    Dit is de tekst zoals je hem ziet op de website.
    In dit geval het woord "robots".
